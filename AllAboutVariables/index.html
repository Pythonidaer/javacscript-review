What are Variables?

Variables are containers for values.

Block Scoping

In JavaScript, block scopes are created by different control structures and declarations that define blocks of code. here are some examples of different block scopes within JavaScript:

If Statement:


        if (true) {
          // This is a block scope created by the if statement
          const message = "Hello, block scope!";
          console.log(message); // Outputs "Hello, block scope!"
        }
        
        // console.log(message); // This would result in an error because 'message' is not accessible here.

For Loop:


        for (let i = 0; i < 3; i++) {
            // This is a block scope created by the for loop
            console.log(i); // Outputs 0, 1, 2
          }
          
          // console.log(i); // This would result in an error because 'i' is not accessible here.

While Loop:


        let count = 0;
        while (count < 3) {
            // This is a block scope created by the while loop
            console.log(count); // Outputs 0, 1, 2
            count++;
        }

        // console.log(count); // This would result in an error because 'count' is not accessible here.

Functions:


        function greet() {
            // This is a block scope created by the function
            const greeting = "Hello, function block!";
            console.log(greeting); // Outputs "Hello, function block!"
          }
          
          greet();
          // console.log(greeting); // This would result in an error because 'greeting' is not accessible here.

ES6 Block with Let/Const Declarations


        {
            // This is a block scope created by the curly braces
            let x = 10;
            const y = 20;
            console.log(x + y); // Outputs 30
          }
          
          // console.log(x); // This would result in an error because 'x' is not accessible here.
          // console.log(y); // This would result in an error because 'y' is not accessible here.

Try-Catch Block:


        try {
            // This is a block scope created by the try block
            const result = 10 / 0; // This will throw an error (division by zero)
            console.log(result); // This will not be executed
          } catch (error) {
            // This is a block scope created by the catch block
            console.error("Error occurred:", error); // Outputs "Error occurred: [Error message]"
          }
          
          // console.log(result); // This would result in an error because 'result' is not accessible here.
          // console.log(error); // This would result in an error because 'error' is not accessible here.
