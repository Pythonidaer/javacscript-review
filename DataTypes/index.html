There are several Primitive Data Types:

  • string
  • undefined
  • number
  • bigint
  • boolean
  • null
  • Symbol

Non-Primitve Data Types

  • Array
  • Object

The above primitives are immutable, and the non-primitve data ones are mutable

new

The `new` keyword is used to create an instance of a user-defined object or a built-in constructor function.
