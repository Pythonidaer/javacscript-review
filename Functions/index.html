Functions are first-class citizens in JavaScript, meaning they can be treated just like any other value, such as numbers, strings, or objects.

This property allows functions to be stored in variables, passed as arguments to other functions, returned from functions, and much more.

This concept is known as "higher-order functions" and is a powerful feature that enables functional programming paradigms
